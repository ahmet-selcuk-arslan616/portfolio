ML-Ready Data Preparation and Cleaning App

Executive Summary:

Using Python, Streamlit, and a full preprocessing pipeline built with pandas and scikit-learn tools, I developed an interactive app that turns raw tabular data into a clean, consistent, and model-ready dataset. The app lets users upload files, detect data types automatically, handle missing values, remove duplicates, engineer new features, optionally treat outliers, and download the cleaned output in the same format. It is designed to reduce manual preprocessing time and make machine learning workflows faster and more reliable.

  • Automatic data type detection
  • Missing value handling and imputation
  • Feature engineering for tabular ML
  • Export-ready cleaned datasets
  • EDA report generation

Business Problem:

Raw business data is often messy and inconsistent. Missing values, duplicate rows, mixed data types, invalid column names, and weak feature structures can slow down analysis and reduce model performance. Preparing these datasets by hand takes time and creates room for error. This app solves that problem by giving users a simple interface where they can upload a dataset and quickly convert it into a machine-learning-ready table without writing preprocessing code from scratch.

Methodology:

  1. Upload tabular data from CSV, Excel, Parquet, or SQLite.
  2. Automatically clean column names, remove duplicates, and standardize missing-value patterns.
  3. Infer column types such as numeric, categorical, and datetime.
  4. Optionally impute missing numeric values with a custom MissForest implementation based on Random Forests.
  5. Optionally generate new features such as math features, relative features, datetime features, and cyclical features.
  6. Optionally apply one-hot encoding, normalization, and outlier removal.
  7. Preview the cleaned dataset and download it in the original file format.

Skills:

Python: data cleaning, preprocessing pipelines, feature engineering, custom transformer development, model-ready dataset preparation, file handling, app logic
Streamlit: interactive UI design, file upload workflows, session-state management, downloadable outputs, EDA preview integration
Machine Learning / Data Preparation: missing value imputation, outlier detection, categorical encoding, scaling, datetime transformation, duplicate and constant-feature removal
Data Tools & Libraries: pandas, NumPy, scikit-learn, PyJanitor, SciPy, SQLite

Results & Business Recommendation:

This app makes tabular data preparation much faster and more accessible, especially for users who want a practical preprocessing workflow without building everything manually. Instead of spending time writing repeated cleaning code, users can upload a dataset, choose a few settings, and get back a cleaner version that is easier to use for machine learning or analysis.

The app supports both a simple automatic mode and a more flexible advanced mode. This makes it useful for both quick cleaning tasks and more controlled preprocessing workflows. It also keeps the experience practical by allowing users to preview outputs, generate an optional EDA report, and download the cleaned dataset in the same format they uploaded.

Because the pipeline combines cleaning, imputation, feature engineering, and export in one place, it helps reduce preprocessing friction and creates a smoother path from raw data to modeling.
